# Regras de Trabalho do Codex / Codex Working Rules

## Fluxo padrão / Default flow
- Sempre começar em modo de planejamento.
- Antes de editar código, apresentar um plano curto (3-6 passos).
- Só executar alterações após minha confirmação explícita: "pode executar".
- Always start with a planning step.
- Before editing code, present a short plan (3-6 steps).
- Only execute changes after my explicit confirmation: "you can proceed".

## Modo chat / Chat mode
- Quando a pergunta for conceitual, responder sem executar comandos e sem editar arquivos.
- Se houver dúvida sobre executar algo, perguntar antes.
- For conceptual questions, respond without running commands and without editing files.
- If there is any doubt about executing actions, ask first.

## Segurança / Safety
- Não instalar dependências nem rodar comandos destrutivos sem confirmação.
- Do not install dependencies or run destructive commands without confirmation.
